# Governance

## Maintainers

TythanAI is maintained by **TythanAI Labs**. Current core maintainers approve all merges to `main`.

## Decision Making

- **Bug fixes and documentation:** any maintainer can merge after one approval.
- **New features:** require issue discussion + one maintainer approval.
- **Breaking changes / architecture:** require maintainer consensus and a 7-day comment period on the issue.
- **Licence or governance changes:** require written agreement from all active maintainers.

## Becoming a Contributor

Open a pull request. See [CONTRIBUTING.md](CONTRIBUTING.md). There is no formal committer ladder yet; as the project grows, recurring contributors will be invited to maintainer status.
