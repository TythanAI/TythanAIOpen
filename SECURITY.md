# Reporting Security Issues in TythanAI

**Do not open a public GitHub issue for security vulnerabilities.**

Email **security@tythanai.io** with:
- Description of the issue and affected component
- Steps to reproduce or proof-of-concept (if safe to share)
- Potential impact assessment

We aim to acknowledge reports within 48 hours and provide a fix timeline within 7 days for critical issues.

## Scope

In scope: the TythanAI platform itself — scanner logic, API server, CLI, and dependencies bundled in this repository.

Out of scope: findings produced *by* TythanAI against third-party code, social engineering, physical access, or DDoS.

## Disclosure Policy

We follow coordinated disclosure. We will work with you to agree a public disclosure timeline (typically 90 days). We credit reporters in release notes unless anonymity is requested.

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest on `main` | ✓ |
| previous minor    | security fixes only |
| older             | ✗ |
